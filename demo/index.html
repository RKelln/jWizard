jWizard A jQuery UI Widget for all your Wizard Form Needs!

Demonstrations

All Configuration Options
Here, all the configuration options are set to their respective default values. While this looks the same as the Default configuration, it allows you to see all the available configuration just by using 'View Source'.
Default Configuration
This is what you will get with absolutely no configuration options. While this itself is not entirely useful, it does demonstrate what is available 'out of the box'.
Navigation Menu Enabled
This is a demonstration of the available side menu, which you can display to your users to give visual feedback of their progress.
Step Counter Enabled
The Step Counter is another way to give your users feedback on their progress. You can use a counter or a percentage, as well as include a jQuery UI Progressbar.
jQuery UI Effects Enabled
jQuery UI includes a great set of transition effects, which I leverage in jWizard as well if you would like the additional eye-candy and visual feedback for your users.
Custom Events
By using these custom event handlers and callbacks, you can add a great deal of complexity and sophistication to your form-filling procedures.
Public Methods
jWizard exposes some methods to you in order to allow you to programmatically interact with the Wizard after being created.
Multiple Instances
The internals of jWizard work in such a way that multiple instances of jWizard will not interfere with one another.
Inside a jQuery UI Dialog
jWizard works just as easily in a jQuery UI Dialog.
